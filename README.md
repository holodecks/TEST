# 助産師による女性の健康相談サイト

経験豊富な助産師が、妊娠・出産・産後ケア・女性の健康に関する相談にお答えするウェブサイトです。

## 機能

- **トップページ**: サービス紹介と主な機能の概要
- **助産師紹介**: 経験豊富な助産師のプロフィール
- **相談フォーム**: オンラインで健康相談を送信
- **健康情報**: 女性の健康に役立つ記事とFAQ
- **お問い合わせ**: 連絡先情報と営業時間

## 技術スタック

- **バックエンド**: Flask 3.0.0
- **フォーム**: Flask-WTF
- **フロントエンド**: HTML5, CSS3
- **バリデーション**: email-validator

## セットアップ方法

### 必要な環境

- Python 3.8以上

### インストール

1. リポジトリをクローン:
```bash
git clone <リポジトリURL>
cd TEST
```

2. 依存パッケージをインストール:
```bash
pip install -r requirements.txt
```

### アプリケーションの起動

```bash
python app.py
```

ブラウザで `http://localhost:5000` にアクセスしてください。

## 機能詳細

### 相談フォーム

ユーザーは以下の情報を入力して相談を送信できます：
- お名前（必須）
- メールアドレス（必須）
- 年齢（任意）
- 相談カテゴリー（選択式）
- 相談内容（必須）

送信された相談は、アプリケーション内で管理され、担当の助産師が24時間以内に対応します。

### セキュリティ

- CSRFトークンによるフォーム保護
- メールアドレスの検証
- 個人情報の厳重な管理
